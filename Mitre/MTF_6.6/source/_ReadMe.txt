This code is compiled with LIBTIFF.  Once libtiff is installed
compile the code in this directory using 'make'.  
If necessary add additional LDFLAGS to identify library locations.
TIFF requirement/capabilities can be excluded by editing the Makefile.

Open a terminal and 
> cd <this-dir>
> make 

Included precompiled executables (in ../Execs)
* macmtf     compiled via gcc for Mac OS X/Intel (compiled on Mavericks) 
* winmtf.exe complied via Min_GW gcc for Windows (compiled on Win7)
include minimal TIFF support(JPEG not included).
